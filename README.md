# Rock, Paper, Scissor Game

This is a simple web-based Rock, Paper, Scissor game implemented using HTML, CSS, and JavaScript. It allows users to play against the computer for a best of 3 rounds match.

## How to Play

1. Clone the repository or download the source code.
2. Open the `index.html` file in your web browser.
3. Click on either the Rock, Paper, or Scissor image to make your selection.
4. The computer will randomly select its choice.
5. The outcome of the round will be displayed, and the scores will be updated accordingly.
6. The first player to reach 3 points wins the match.
7. Click "Play Again" to reset the game and start a new match.

## Features

- Simple and intuitive user interface.
- Best of 3 rounds match structure.
- Visual representation of player and computer choices.
- Responsive design for desktop and mobile devices.
